### Contenido principal

Este es el sumatorio de una [[Sucesión geométrica]]:

$S_n = a_1 + a_1 · r + a_1 · r^2 + \dots + a_1 · r^{n-1}$
$rS_n =    a_1 · r + a_1 · r^2 + \dots + a_1 · r^{n-1} + a_1 · r^n$
$(1-r)S_n = a_1 - a_1 · r^n$

En conclusión:
### $S_n = \frac{a_1 - a_1 · r^{n-1} · r}{1-r} = \frac{a_1 - a_nr}{1-r}$
