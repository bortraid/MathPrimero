```ad-formal
title: Formal definition
Dado un [[intervalo]] $[a,b]$, una partición de $[a,b]$ es un conjunto finito $\{x_0, x_1, \dots, x_n \}$, donde $a = x_0 < x_1 < x_2 < \dots < x_n = b$.
```
