### Contenido principal

Estas son las propiedades del valor absoluto:
1. $|x| \ge 0$ y $|x| = 0 \iff x = 0$
2. $|x+y| \le |x| + |y|$.
3. $|xy| = |x||y|$.
4. $|x| = |-x|$.
5. $|x| \le y \iff -y \le x \le y$.
6. $||x| - |y|| \le |x-y|$.
7. $|x^{-1}| = |x|^{-1}, x \not = 0$.
