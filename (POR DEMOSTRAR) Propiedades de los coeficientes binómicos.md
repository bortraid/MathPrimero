```ad-proposition
$${n \choose k} = {n \choose n-k} \textrm{ para } n \ge 1 \textrm{ y } 0 \le k \le n$$
$${n \choose k} = {n-1 \choose k} + {n-1 \choose k-1} \textrm{ para } n \ge 2 \textrm{ y } 1 \le k \le n-1$$
```


```ad-proof
Triángulo de Pascal.
```
